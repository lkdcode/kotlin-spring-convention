# framework/storage/s3 — AWS S3 Pre-signed URL

`PresignedStoragePort` 구현체. 서버는 URL 발급만 — 클라이언트가 S3에 직접 업로드.

> `AwsS3Properties`, `AwsS3Service` → `cloud/aws/s3/s3.md` 참조 (단일 정의)

## 흐름

```
1. Client --[POST /api/files/put-url?key=...]--> Server
                                             <-- S3 Pre-signed PUT URL

2. Client --[PUT {presignedUrl} + 파일]--> S3 (서버 비개입)
                                       <-- 완료

3. Client --[POST /api/foos + key]--> Server (비즈니스 처리)
```

## AwsS3StorageAdapter

```kotlin
@Component
class AwsS3StorageAdapter(
    private val properties: AwsS3Properties,
) : PresignedStoragePort {

    override fun putUrl(key: String): URL =
        s3Presigner.presignPutObject(
            PutObjectPresignRequest.builder()
                .putObjectRequest(
                    PutObjectRequest.builder()
                        .bucket(properties.bucketName)
                        .key(key)
                        .build()
                )
                .signatureDuration(properties.putExpired)
                .build()
        ).url()

    override fun getUrl(key: String): URL? {
        if (key.isBlank()) return null
        return s3Presigner.presignGetObject(
            GetObjectPresignRequest.builder()
                .getObjectRequest(
                    GetObjectRequest.builder()
                        .bucket(properties.bucketName)
                        .key(key)
                        .build()
                )
                .signatureDuration(properties.getExpired)
                .build()
        ).url()
    }

    override fun delete(key: String) {
        s3Client.deleteObject {
            it.bucket(properties.bucketName).key(key)
        }
    }

    override fun copy(sourceKey: String, destKey: String) {
        s3Client.copyObject(
            CopyObjectRequest.builder()
                .sourceBucket(properties.bucketName).sourceKey(sourceKey)
                .destinationBucket(properties.bucketName).destinationKey(destKey)
                .build()
        )
    }

    private val provider = StaticCredentialsProvider.create(
        AwsBasicCredentials.create(properties.accessKeyId, properties.secretAccessKey)
    )

    private val s3Presigner: S3Presigner = S3Presigner.builder()
        .region(properties.bucketRegion)
        .credentialsProvider(provider)
        .endpointOverride(properties.endPointUri)
        .build()

    private val s3Client: S3Client = S3Client.builder()
        .region(properties.bucketRegion)
        .credentialsProvider(provider)
        .endpointOverride(properties.endPointUri)
        .build()
}
```

## 응답 모델

```kotlin
data class PresignedPutResult(
    val putUrl: String,
    val key: String,
)

data class PresignedGetResult(
    val getUrl: String,
    val key: String,
)
```

## 규칙

- 서버가 파일을 직접 수신하지 않음 — URL 발급만
- `getUrl` 은 key 가 blank 면 `null` 반환
- Pre-signed URL 만료 시간은 설정으로 관리 — 하드코딩 금지
- `copy` — 임시 업로드 → 정식 경로 이동 시 사용
- Local Pre-signed 와 동일한 `PresignedStoragePort` 구현 — `@Profile` 로 환경별 교체
