# cloud/aws/s3 — AWS S3

Pre-signed URL 발급 전용. 서버가 파일을 직접 처리하지 않음.
`framework/storage/s3/` 의 `AwsS3StorageAdapter` 가 이 서비스를 사용.

## AwsS3Properties

```kotlin
@Component
data class AwsS3Properties(
    @Value("\${aws.app.access_key_id}") val accessKeyId: String,
    @Value("\${aws.app.secret_access_key}") val secretAccessKey: String,
    @Value("\${aws.app.bucket_name}") val bucketName: String,
    @Value("\${aws.app.region}") private val region: String,
    @Value("\${aws.presigned.get_expired}") private val getExpiredSeconds: String,
    @Value("\${aws.presigned.put_expired}") private val putExpiredSeconds: String,
) {
    val bucketRegion: Region get() = Region.of(region)
    val getExpired: Duration get() = Duration.ofSeconds(getExpiredSeconds.toLong())
    val putExpired: Duration get() = Duration.ofSeconds(putExpiredSeconds.toLong())
}
```

## AwsS3Service

```kotlin
@Component
class AwsS3Service(
    private val properties: AwsS3Properties,
) {
    fun generatePresignedPutUrl(key: String): URL =
        s3Presigner.presignPutObject(
            PutObjectPresignRequest.builder()
                .putObjectRequest(
                    PutObjectRequest.builder()
                        .bucket(properties.bucketName)
                        .key(key)
                        .build()
                )
                .signatureDuration(properties.putExpired)
                .build()
        ).url()

    fun generatePresignedGetUrl(key: String): URL =
        s3Presigner.presignGetObject(
            GetObjectPresignRequest.builder()
                .getObjectRequest(
                    GetObjectRequest.builder()
                        .bucket(properties.bucketName)
                        .key(key)
                        .build()
                )
                .signatureDuration(properties.getExpired)
                .build()
        ).url()

    private val provider = StaticCredentialsProvider.create(
        AwsBasicCredentials.create(properties.accessKeyId, properties.secretAccessKey)
    )

    private val s3Presigner: S3Presigner = S3Presigner.builder()
        .region(properties.bucketRegion)
        .credentialsProvider(provider)
        .build()
}
```

## 규칙

- `AwsS3Service` 는 AWS SDK 직접 호출만 — 비즈니스 로직 없음
- `framework/storage/` 어댑터를 통해서만 사용 — `domains/` 에서 직접 import 금지
- Pre-signed URL 만료 시간은 설정으로 관리 — 하드코딩 금지
